# Jarvis AI – Your Personal Assistant

Wake word: **"Hey Jarvis"**  
Voice control, internet brain, PC automation, hand gestures.

## Setup

1. Install Python 3.10+
2. Clone this repo
3. Run `pip install -r requirements.txt`
4. (Optional) Add OpenAI API key in `config.py` for smarter answers
5. Run `python main.py`

## Features
- Offline wake word (Porcupine)
- Voice commands
- YouTube, scroll, open website/folder, screenshot
- Internet search (DuckDuckGo free)
- Memory across sessions
- Gesture detection (coming soon)

## Requirements
- Microphone
- Webcam (for gestures)
- Windows (Linux/Mac partial support)
